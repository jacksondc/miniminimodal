Mini Mini Modal

Mini Mini Modal is a tiny JS modal with no dependencies. Probably the tiniest; it's basically just alert() but it looks nicer. Trigger with mmm('message').

Options

You can also pass an options object:

          
          mmm('Lorem Ipsum', {
            closeText: 'Close',
            raw: true,
            transitionDuration: 2
          });

closeText

Specifies the text to use on the close button. Defaults to “OK”.

raw

If true, your message will be treated as raw html, won't be wrapped in a paragraph tag. Defualts to false.

transitionDuration

Specifies how long to wait before removing the modal DOM elements after close, or Defaults to 0.5.
